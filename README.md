# CS-test-Bigmodel
⭐ 项目简介 ：Try to build up a 大模型  
## 项目介绍 
依据豆包总结，要自建大模型，只需要 4 样东西：  
1. 腾讯云 T4 GPU 服务器  
2. CFS 文件存储  
3. 开源基座模型（ChatGLM3-6B / Qwen-7B）  
4. 你的企业业务数据（1000 条以上）

### 最终「最便宜能跑」套餐  
GPU 服务器：GN7.T4.XLARGE16  
4 核 16G / T4 16G / 硬盘 100G  
无 CFS  
无 COS  
包月 ≈ 600～750 元  

### 推荐配置（稳、不贵、能生产）
GPU 服务器：GN7.T4.2XLARGE32  
8 核 32G / T4 16G / 硬盘 200G  
无 CFS  
无 COS  
包月 ≈ 1000～1300 元

想最便宜：选 4 核 16G T4，每月 600～750 元，能跑 7B 微调 + 部署。  
想最稳：选 8 核 32G T4，每月 1000～1300 元，不折腾、不爆显存。  
CFS、COS 你现在都可以不买，能省 100 元 / 月。
